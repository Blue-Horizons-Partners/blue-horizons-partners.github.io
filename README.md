# Blue Horizons Partners Website

Official website for Blue Horizons Partners - Strategic Partnerships & Innovative Solutions

## 🚀 Minimal Mistakes Theme Migration

This repository contains documentation and plans for migrating to the [Minimal Mistakes Jekyll theme](https://github.com/mmistakes/minimal-mistakes).

We've prepared comprehensive guides to help with the theme implementation:

1. **[Implementation Plan](./DOCUMENTS/MINIMAL_MISTAKES_IMPLEMENTATION_PLAN.md)** - Complete step-by-step migration guide
2. **[Image Requirements](./DOCUMENTS/IMAGE_REQUIREMENTS.md)** - Detailed image specifications and dimensions
3. **[Quick Reference](./DOCUMENTS/QUICK_REFERENCE.md)** - Essential info and cheat sheet
4. **[Visual Layout Guide](./DOCUMENTS/VISUAL_LAYOUT_GUIDE.md)** - Visual representation of layouts and image placement

### ⚡ Quick Start

1. Review the [Quick Reference](./DOCUMENTS/QUICK_REFERENCE.md) for essential information
2. Gather images according to [Image Requirements](./DOCUMENTS/IMAGE_REQUIREMENTS.md)
3. Follow the [Implementation Plan](./DOCUMENTS/MINIMAL_MISTAKES_IMPLEMENTATION_PLAN.md)

### 📋 Minimum Required Images

To launch the site, you'll need these 5 images:

| Image        | Dimensions | Purpose              |
| ------------ | ---------- | -------------------- |
| Logo         | 88×88px    | Site header          |
| Bio Photo    | 200×200px  | Author profile       |
| OG Image     | 1200×630px | Social media sharing |
| Hero Banner  | 1280×720px | Homepage background  |
| About Header | 1280×720px | About page header    |

See [Image Requirements](./DOCUMENTS/IMAGE_REQUIREMENTS.md) for complete details and optional images.

### 🎨 Theme Configuration

The Minimal Mistakes theme provides:

- ✅ Professional, responsive design
- ✅ Multiple layout options (splash, single, archive, home)
- ✅ Built-in SEO optimization
- ✅ Social media integration
- ✅ **Selected Skin:** `default` (clean, professional look)
- ✅ Author profiles and sidebars
- ✅ Google Analytics support
- ✅ Comments integration options
- ✅ Search functionality

### 🛠️ Local Development

```bash
# Install dependencies
bundle install

# Run local server
bundle exec jekyll serve

# View site at: http://localhost:4000
```
