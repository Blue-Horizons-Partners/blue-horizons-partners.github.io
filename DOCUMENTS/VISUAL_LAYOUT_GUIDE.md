## Image Hierarchy & Priority

### Critical Path (Can't Launch Without):

```
1. Logo (88×88px)
   └─> Appears on every single page in header

2. Bio Photo (200×200px)
   └─> Shows in sidebar on About and blog posts

3. OG Image (1200×630px)
   └─> Shows when site is shared on social media

4. Hero Banner (1280×720px)
   └─> First impression on homepage
```

### Important (Should Have for Professional Look):

```
5. About Header (1280×720px)
   └─> Makes About page more engaging

6. Services Header (1280×720px)
   └─> Professional look for Services page

7. Feature Images (600×400px × 3)
   └─> Visual interest on homepage
```

### Optional (Can Add Later):

```
8. Contact Header (1280×720px)
   └─> Can use solid color overlay instead

9. Team Photos (300×300px each)
   └─> Only needed if showcasing team

10. Blog Images (500×300px each)
    └─> Only needed if you have blog posts
```

## Color Overlay Examples

Instead of using images for every header, you can use color overlays:

### Solid Color Overlay:

```yaml
header:
  overlay_color: "#5e616c"  # Dark gray
  # or
  overlay_color: "#1976D2"  # Blue (matches "Blue Horizons")
```

### Image with Color Filter:

```yaml
header:
  overlay_image: /assets/images/hero-banner.jpg
  overlay_filter: "0.5"  # 50% dark overlay
  # or
  overlay_filter: "rgba(25, 118, 210, 0.5)"  # Blue overlay
```

**Benefit:** Reduces number of images needed while maintaining professional look.

## Tips for Choosing/Creating Images

### Hero Banner (Homepage):

✅ **Good choices:**

- Skyline with clouds (matches "horizons")
- Business handshake with bokeh background
- Modern office space with natural light
- Abstract blue gradient (simple, on-brand)

❌ **Avoid:**

- Busy images with lots of small details
- Very dark images (text becomes hard to read)
- Images with text already on them
- Low resolution or pixelated images

### Feature Block Images:

✅ **Good choices:**

- Icons or illustrations on solid backgrounds
- Simple, clear photographs
- Consistent style across all three
- Related to the service/feature

❌ **Avoid:**

- Random stock photos
- Different styles mixing (photo + illustration)
- Too much detail that gets lost when resized

### Professional Tips:

1. **Consistency:** Use same style across all images
2. **Contrast:** Ensure text remains readable on overlays
3. **Brand colors:** Incorporate your brand colors subtly
4. **White space:** Images with breathing room work better
5. **People:** If using people, ensure diverse, professional representation

## Testing Your Images

After adding images, test:

```
Desktop View (1920px):
- [ ] Hero image fills screen nicely
- [ ] Text overlay is readable
- [ ] Feature images are crisp
- [ ] Logo is clear

Tablet View (768px):
- [ ] Images resize appropriately
- [ ] Text remains readable
- [ ] Feature blocks stack or shrink well

Mobile View (375px):
- [ ] Hero image shows key elements
- [ ] Text overlay doesn't overflow
- [ ] Images load quickly
- [ ] Logo is visible

Performance:
- [ ] Page loads in < 3 seconds
- [ ] Images are compressed
- [ ] No broken image links
```

## Quick Start: 5-Minute Placeholder Setup

If you want to launch quickly with placeholder images:

1. **Logo:** Use text-based logo initially
2. **Bio Photo:** Use company logo or generic business icon
3. **OG Image:** Create in Canva with logo + text
4. **Hero Banner:** Find one good business image from Unsplash
5. **Headers:** Use solid color overlays instead of images

Then improve images over time as you get professional photography/design work done.
